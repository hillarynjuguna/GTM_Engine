# GTM Readiness Scorecard

## Purpose

Track progress from **"conceptually complete"** to **"operationally ready"** across 5 layers. Each layer must turn GREEN before scaling investment.

## Weekly Review Protocol

Every Friday:
1. Update each criterion with current status
2. Calculate layer scores
3. Identify top 3 blockers
4. Define next week's priorities
5. Record in `weekly_reviews/` folder

### Kill / Scale Decision Framework
| Signal | Decision |
|---|---|
| Layer 1–3 all GREEN | → Scale acquisition budget 2–3x |
| Layer 3 RED | → STOP acquisition, fix onboarding |
| Layer 4 RED | → STOP scaling, fix product/value |
| CAC > 2× target for 3 weeks | → Pivot channel or messaging |
| Activation < 30% for 2 weeks | → Redesign onboarding completely |
