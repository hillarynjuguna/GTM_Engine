# Acquisition Funnels — 3 Validated Channels

## Funnel 1: Meta Ads → WhatsApp (Primary)

### Architecture
```
Facebook/Instagram Ad → Click-to-WhatsApp → Automated Qualification → Human-Assisted Close → Onboarding
```

### Targeting Spec
| Parameter | Value |
|---|---|
| **Platform** | Facebook + Instagram |
| **Objective** | Messages (Click-to-WhatsApp) |
| **Location** | Klang Valley, Penang, JB, Ipoh, Melaka, Kota Kinabalu |
| **Age** | 25–55 |
| **Interests** | Restaurant owner, F&B business, small business, POS system, food delivery |
| **Behaviors** | Small business owners, page admins |
| **Language** | Malay, English |
| **Exclusions** | Corporate chains, franchise HQs |

### Budget & Economics
| Metric | Target |
|---|---|
| Daily budget | RM 100–200 |
| Monthly budget | RM 3,000–6,000 |
| CPM (est.) | RM 15–30 |
| CTR (est.) | 1.5–3% |
| **CPL** | **RM 20–40** |
| WhatsApp → Demo | 40–60% |
| Demo → Close | 30–50% |
| **CAC** | **RM 100–250** |

### Ad Variants (rotate 3 at minimum)
1. Compliance-led (fear/urgency)
2. Revenue-led (gain/opportunity)
3. Simplicity-led (ease/speed)

### Success Criteria
- CPL < RM 50
- Reply rate > 40%
- Close rate > 20%

## Funnel 2: Direct WhatsApp Outreach (Warm)

### Architecture
```
Curated SME List → Personalized WhatsApp Message → Qualification → Demo → Close → Onboarding
```

### Lead Sources
| Source | Method |
|---|---|
| Google Maps scraping | F&B businesses in target cities |
| Facebook Groups | "Malaysian restauran owners", "Peniaga F&B Malaysia" |
| Industry associations | SME Corp, MRCA (restaurant & café association) |
| Supplier networks | F&B ingredient distributors, packaging companies |
| Referrals | Existing user introductions |

### Outreach Cadence
| Day | Action |
|---|---|
| Day 1 | Initial personalized message |
| Day 2 | Follow-up if no reply (softer angle) |
| Day 5 | Value-add message (compliance tip/resource) |
| Day 7 | Final CTA (demo offer) |

### Economics
| Metric | Target |
|---|---|
| Messages sent/day | 30–50 |
| Reply rate | 15–25% |
| Demo conversion | 40–60% |
| **CPL** | **RM 5–15** (labor only) |
| **CAC** | **RM 30–80** |
| Time cost | 2–3 hours/day |

### Success Criteria
- Reply rate > 15%
- Demo booking > 10% of outreach
- Close rate > 30%

## Funnel 3: Landing Page → Demo Request (Inbound)

### Architecture
```
Google/Social Traffic → Landing Page → Demo Request Form → WhatsApp Follow-Up → Demo → Close → Onboarding
```

### Traffic Sources
| Source | Strategy |
|---|---|
| **SEO** | Target: "e-invoicing malaysia", "LHDN e-invoice SME", "sistem invois F&B" |
| **Google Ads** | Keywords: "e-invoice software malaysia", "WhatsApp order system" |
| **Social organic** | Educational content about compliance deadlines |
| **Referral** | User share links with incentive |

### Landing Page Requirements
- Hero: compliance hook + clear value promise
- Social proof: user count, testimonials, compliance badge
- 3 value props with icons
- Demo video or animation
- WhatsApp CTA (primary) + form CTA (secondary)
- Mobile-first design
- < 3 second load time

### Economics
| Metric | Target |
|---|---|
| Monthly visitors | 500–2,000 |
| Conversion rate | 5–10% |
| **CPL** | **RM 30–60** (blended) |
| Demo → Close | 30–50% |
| **CAC** | **RM 80–200** |

### Success Criteria
- Page conversion > 5%
- Bounce rate < 60%
- Form completion > 70% of those who start
